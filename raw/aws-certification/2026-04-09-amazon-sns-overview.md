# What is Amazon SNS?
> Source: https://docs.aws.amazon.com/sns/latest/dg/welcome.html
> Collected: 2026-04-09
> Published: Unknown

Amazon Simple Notification Service (Amazon SNS) is a fully managed service that provides message delivery from publishers to subscribers.

Publishers send messages to a topic, which acts as a communication channel for multiple subscribers.

Subscribers can receive messages through endpoints such as:

- Amazon SQS
- AWS Lambda
- HTTP or HTTPS endpoints
- Email
- Mobile push notifications
- SMS

Common Amazon SNS scenarios include:

- Application-to-application fanout messaging
- Application and system alerts
- User notifications
- Mobile push notifications

Amazon SNS is designed for asynchronous communication and broad message distribution.

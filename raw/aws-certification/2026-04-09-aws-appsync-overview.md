# What is AWS AppSync?
> Source: https://docs.aws.amazon.com/appsync/latest/devguide/what-is-appsync.html
> Collected: 2026-04-09
> Published: Unknown

AWS AppSync enables developers to connect applications and services to data and events with secure, serverless, and high-performing GraphQL and Pub/Sub APIs.

You can use AWS AppSync to:

- Access data from one or more data sources from a single GraphQL API endpoint
- Combine multiple source GraphQL APIs into a single merged GraphQL API
- Publish real-time data updates to applications
- Use built-in security, monitoring, logging, and tracing
- Use optional caching for low latency

AWS AppSync features include:

- GraphQL-based querying
- Serverless WebSockets for GraphQL subscriptions and pub/sub channels
- Server-side caching
- JavaScript and TypeScript support for business logic
- Private APIs and AWS WAF integration
- Authorization with API keys, IAM, Amazon Cognito, OpenID Connect, and Lambda authorization
- Merged APIs for federated use cases

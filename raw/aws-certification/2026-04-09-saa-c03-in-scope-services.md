# In-scope AWS services and features
> Source: https://docs.aws.amazon.com/aws-certification/latest/solutions-architect-associate-03/saa-03-in-scope-services.html
> Collected: 2026-04-09
> Published: Unknown

The following list contains AWS services and features that are in scope for the exam. This list is non-exhaustive and is subject to change. AWS offerings appear in categories that align with the offerings' primary functions:

## Topic categories

- Analytics
- Application Integration
- AWS Cost Management
- Compute
- Containers
- Database
- Developer Tools
- Front-End Web and Mobile
- Machine Learning
- Management and Governance
- Media Services
- Migration and Transfer
- Networking and Content Delivery
- Security, Identity, and Compliance
- Serverless
- Storage

## Selected in-scope services

- Analytics: Amazon Athena, Amazon EMR, AWS Glue, Amazon Kinesis, AWS Lake Formation, Amazon Redshift
- Application Integration: Amazon EventBridge, Amazon SNS, Amazon SQS, AWS Step Functions
- Cost Management: AWS Budgets, AWS Cost and Usage Report, AWS Cost Explorer, Savings Plans
- Compute and Containers: Amazon EC2, Amazon EC2 Auto Scaling, AWS Batch, Amazon ECS, Amazon EKS, AWS Fargate
- Database: Amazon Aurora, Amazon DynamoDB, Amazon ElastiCache, Amazon RDS, Amazon Redshift
- Management and Governance: AWS CloudFormation, AWS CloudTrail, Amazon CloudWatch, AWS Config, AWS Control Tower, AWS Organizations, AWS Trusted Advisor, AWS Well-Architected Tool
- Networking and Content Delivery: Amazon CloudFront, AWS Direct Connect, Elastic Load Balancing (ELB), AWS Global Accelerator, Amazon Route 53, AWS Transit Gateway, Amazon VPC
- Security, Identity, and Compliance: AWS Certificate Manager (ACM), Amazon Cognito, Amazon GuardDuty, AWS IAM Identity Center, AWS KMS, Amazon Macie, AWS Secrets Manager, AWS Shield, AWS WAF, IAM
- Serverless: AWS AppSync, AWS Fargate, AWS Lambda
- Storage: AWS Backup, Amazon EBS, Amazon EFS, Amazon FSx, Amazon S3, Amazon S3 Glacier, AWS Storage Gateway

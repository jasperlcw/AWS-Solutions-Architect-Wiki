# AWS Solutions Architect Wiki (SAA-C03)
This repo contains markdown files, sources using a skill created from [Karpathy's LLM Wiki](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f) gist. This idea of this is to gather relevant AWS docs for the SAA-C03 exam, and to gather relevant practice exam questions in hopes that an AI agent can generate new practice questions and material to aid in the study of the certifcation exam.

## How to use
Copy and paste the LLM Wiki idea from the Karpathy gist linked above, and ask your AI Agent to generate a Skill. Alternatively, you can use an already generated Skill file like the one from the [Astro-Han repository](https://github.com/Astro-Han/karpathy-llm-wiki).

Once the skill is set up, ingest AWS documents by prompting the agent to ingest a link, followed by the actual link:
`Ingest this into the LLM wiki: https://docs.aws.amazon.com/aws-certification/latest/solutions-architect-associate-03/solutions-architect-associate-03.html`

Use this idea to create a strong knowledge base, then ask it to generate new practice questions based on concrete practice questions sourced from the web.

# Disclaimer
This repo contains questions and answers that were sourced from the open internet. Official practice questions from AWS or other practice exam providers were not used.
